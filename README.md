# r-and-data-analysis-msc

R programming and data analysis coursework from my Data Science MSc.

## About
This repository contains R programming and data analysis work completed 
as part of my Data Science MSc, including lecture code, workshop exercises, 
lab assignments, and project work. Content spans core R programming through 
to data wrangling, visualisation, and statistical analysis.

## Repository Structure
| Folder | Description |
|--------|-------------|
| `diamonds-dataset-project-seminar-1` | Seminar project analysing the diamonds dataset |
| `lab2` | Lab 2 exercises and work |
| `misc-practice` | Miscellaneous R practice files |
| `r-crash-course` | Introductory R crash course code |
| `r-lab1-assignment` | Lab 1 assignment submission |
| `r-workshop-main` | Workshop exercises and solutions |

## Topics Covered
- Core R basics
- Data wrangling (dplyr, tidyverse)
- Data visualisation (ggplot2)
- Statistical analysis
- Working with real-world datasets
- R Markdown
